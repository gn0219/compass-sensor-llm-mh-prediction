Sensor data-based LLM project for mental health prediction.

## Usage

After configuring the config files, you can run the evaluation with:

```bash
python run_evaluation.py
```

## Config Files

The project uses configuration files in the `config/` directory:
- `prompt_configs.yaml`: Prompt configuration
- `globem_use_cols.json`: GLOBEM data column configuration
- `ces_use_cols.json`: CES data column configuration
- `mentaliot_use_cols.json`: MentalIoT data column configuration
